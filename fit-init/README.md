# 🧘 Fit-Init: Smart AI Self-Defence & Yoga Trainer

[![Python](https://img.shields.io/badge/Python-3.12-blue?style=for-the-badge&logo=python&logoColor=white)](https://www.python.org/)

[![MediaPipe](https://img.shields.io/badge/MediaPipe-Latest-0078D4?style=for-the-badge&logo=google&logoColor=white)](https://mediapipe.dev/)
[![TensorFlow](https://img.shields.io/badge/TensorFlow-2.17-FF6F00?style=for-the-badge&logo=tensorflow&logoColor=white)](https://www.tensorflow.org/)

Fit-Init is a premium AI-powered platform designed to empower women through self-defence training and holistic yoga practice. Using advanced Computer Vision and Machine Learning, the application provides real-time feedback on your poses to ensure maximum safety and effectiveness.

---

## ✨ Key Features

- **🛡️ 4-Level Self-Defence Journey**: Progress from basic warm-ups to advanced defensive techniques.
- **🧘 Real-time Yoga Correction**: AI-guided sessions for classic asanas like Vriksasana and Trikon Asana.
- **📊 Precision Accuracy Tracking**: Live joint-angle analysis with 90-95% detection accuracy.
- **💎 Premium Dark UI**: A sleek, glassmorphic interface designed for focus and immersion.
- **📱 Responsive Experience**: Training accessible anywhere, on any device.

## 🚀 Getting Started

### 1. Installation
Ensure you have Python 3.12 installed, then set up your environment:

```powershell
# Create virtual environment
python -m venv venv

# Activate venv (Windows)
.\venv\Scripts\activate

# Install dependencies
pip install -r requirements_py312.txt
```

### 2. Running the App
Start the Flask development server:

```powershell
python app.py
```

### 3. Accessing the Interface
**⚠️ IMPORTANT:** Do not open the HTML files directly from your folders. Visit the app through your browser at:
👉 **[http://127.0.0.1:5000](http://127.0.0.1:5000)**
